OpenPencil Fork — 改代码 + 打 exe 流程（方案 B）

【日常开发】
  cd D:\code\ResearchScope2\open-pencil-fork
  bun install
  bun run dev
  浏览器 http://localhost:1420 改代码、热更新

【提交并构建 Windows 安装包】
  git add .
  git commit -m "你的改动说明"
  git push origin master

  git tag v0.13.2-build2
  git push origin v0.13.2-build2

  等 GitHub Actions Build 完成（约 15 分钟）
  下载：https://github.com/Ongoinger/open-pencil/releases
  文件：OpenPencil_0.13.2_x64-setup.exe

【安装前清理旧版】
  设置 → 应用 → 卸载 OpenPencil
  删除 %LOCALAPPDATA%\OpenPencil（若存在）

【注意】
  - 只在本目录 open-pencil-fork 改代码，不要用 robocopy 同步
  - 不要用 open-pencil-master 整目录覆盖
  - tag 名每次换新，如 v0.13.2-build3
  - 只看 windows-x64 任务是否成功即可
