OpenPencil Fork — 自己打 exe

【安装前必做】
  卸载旧版 OpenPencil（官方版和 fork 版都卸）
  删除 %LOCALAPPDATA%\OpenPencil
  删除 %LOCALAPPDATA%\com.ongoinger.open-pencil
  fork 使用独立 identifier，与官方 AppData 隔离

【本地打 exe】
  cd D:\code\ResearchScope2\open-pencil-fork
  bun install
  bun run build:packages
  bun run tauri build
  产物: desktop\target\release\bundle\nsis\OpenPencil_0.13.2_x64-setup.exe

【GitHub Actions 构建】
  git add .
  git commit -m "你的改动"
  git push origin master
  git tag v0.13.2-fork2
  git push origin v0.13.2-fork2

【LFS push 失败时】
  git config --local lfs.url "https://github.com/Ongoinger/open-pencil.git/info/lfs"
  git lfs push origin master --all
  git push origin master --force-with-lease
